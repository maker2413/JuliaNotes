Hello world, the most basic of programs, the intro to all languages. In Julia
Hello World Can be achieved with something so simple it resembles Python:
#+name: hello
#+begin_src julia :exports both :results output
  println("Hello World!")
#+end_src

This results in:
#+RESULTS: hello
: Hello World!

Since this lesson would be too short if we just printed a line let's also go
over some of the other basic things to fill this page more. Let's begin with the
simplest of things... Comments!:
#+begin_src julia :results none
  # This is a single line comment.
#+end_src

We also have multi line comments:
#+begin_src julia :results none
  #=
  This is a multi line comment.
  These are made with '#=' and '=#'.
  =#
#+end_src

We should also go over a bit about ~println~. In Julia for basic printing
options we have ~print~ which will just print the string we give it and we also
have ~println~ which prints out the given value and automatically adds a newline
character (~\n~). We can also interpolate strings in Julia with ~$~. To try
this out let's setup some quick variable, which we will discuss more later:
#+name: vars
#+begin_src julia :results none :session interpolation
  # Setting up some variables
  first = "Dave"
  last = "Davidson"
#+end_src

Then we can use those in ~print~ and ~println~ statements:
#+name: interpolation
#+begin_src julia :exports both :results output :session interpolation
  print("First: $first ")
  println("Last: $last")
#+end_src

This would give us:
#+RESULTS: interpolation
: First: Dave Last: Davidson

We can also achieve this by listing items separately with ~print~ or ~println~
as these functions take multiple arguments:
#+printing
#+begin_src julia :exports both :session interpolation
  println("First: ", first, " Last: ", last)
#+end_src

The same output:x
#+RESULTS:
: First: Dave Last: Davidson
